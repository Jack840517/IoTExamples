# IoTことはじめ

本カリキュラムのこのセクションでは、モノのインターネット (Internet of Things) の概要と基本的なコンセプトを学びます。また、クラウドに接続する初めての 'Hello World' IoT プロジェクトに取り組みます。このプロジェクトでは、センサーによって計測された光量が下がると点灯する常夜灯を作ります。
